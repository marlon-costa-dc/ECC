# Frontend Design Review Checklist

Checklist for the [frontend-design-direction](../SKILL.md) skill.

- [ ] First viewport communicates the product, workflow, or object.
- [ ] Visual hierarchy supports scanning and repeated use.
- [ ] Typography fits containers and does not overlap adjacent content.
- [ ] Colors have contrast and avoid a one-note palette.
- [ ] Icons are used for familiar tool actions.
- [ ] Responsive layout has stable dimensions for boards, grids, toolbars, controls, tiles, and counters.
- [ ] Assets carry the subject matter instead of acting as filler.
- [ ] Motion improves orientation and does not mask sluggishness.
- [ ] Result matches existing frontend conventions unless there is a clear reason to depart.
