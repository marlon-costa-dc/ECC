# Deep Research

**UTILITY SKILL**

## USE FOR

- Requests about deep research.
- Workflows described in this skill.
- Operator tasks within this scope.


## DO NOT USE FOR

- questions unrelated to deep-research.
- creating projects or architecture from scratch.


## Workflow

1. Understand the goal. Ask 1-2 clarifying questions if needed.
2. Plan: break the topic into 3-5 sub-questions.
3. Search: run queries with 2-3 keyword variations; aim for reputable sources.


## Critical rules

- Prefer canonical sources.
- Require evidence before claiming success.


## Example

**Input:** a request.
**Output:** a concise response.


## Troubleshooting

- Unclear scope → ask.
- Missing context → state assumptions.
